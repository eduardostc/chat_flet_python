1- Comando: pip install flet

Instalação do Flet que é uma ferramenta do python para criação de sites, aplicativos e programas de computador. 
Flet é uma biblioteca Python que permite criar interfaces gráficas de usuário (GUIs) para sites, aplicativos e programas de desktop de maneira simples e fácil,
usando apenas Python. Ele é uma alternativa mais recente para quem deseja construir aplicativos com interfaces modernas, sem a necessidade de usar HTML, CSS ou JavaScript diretamente.
Com Flet, você pode construir tanto aplicativos web quanto aplicativos desktop usando Python, similar ao que se faz com Flutter.

import flet
